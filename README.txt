Mã nguồn được viết trên Visual Studio 2017
Các chức năng đã thực hiện:
.Nén tất cả các file trong thư mục, nén luôn những file trong thư mục con
.Xem thông tin các file đã nén
.Giải nén tất cả các file
.Giải nén các file được chọn

Lưu ý: 
.Tên file, tên thư mục phải là tiếng việt không dấu
.Các trường hợp mà tên file cần nén quá dài(hơn 70 kí tự, không bao gồm đường dẫn chứa) thì giao diện hiển thị sẽ hơi xấu
.Tốc độ nén/giải nén chưa thực sự tối ưu, nhất là đối với các file multimedia kích thước lớn
Có thể sẽ mất tầm 2 - 3  phút đối với những file multimedia có kích thước từ 100MB
.Khi nhập tên file kết quả cho việc nén file, không cần nhập phần đuôi (đuôi mặc định sẽ là .hfm)
.Khi nhập tên file để giải nén/xem thông tin, nhập thêm phần đuôi .hfm
